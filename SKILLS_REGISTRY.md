# Registre des Compétences et Règles

## Projet : Portaille Captif Microtik
Ce fichier documente les prérequis techniques et les règles spécifiques du projet.

### Compétences (Skills)
- **Skill: MikroTik RouterOS Scripting** : Manipulation des variables `$(...)` et de la logique de login/logout MikroTik.
- **Skill: UI Design (Bootstrap/Modern Web)** : Utilisation de Bootstrap 5 pour le design mobile-first des templates.
- **Skill: Web Development (Next.js/React)** : Pour la création de l'éditeur web "Universal Customizer".
- **Skill: API Development** : Gestion de la génération de fichiers et potentiellement communication avec l'API MikroTik (via RADCLI ou API REST).

### Règles Spécifiques
1. **Sauvegarde Anti-Régression** : Toujours copier les fichiers dans `.CloneBeforeUpdate` avant modification.
2. **Standard de Langue** : Réponses exclusivement en Français.
3. **Validation** : Toujours vérifier la syntaxe HTML et le bon rendu sur mobile.
